# 2026年2月23日 Threads 文章重點整理：Awesome-openclaw-skills

## 標題
🔥兄弟们，你是不是被OpenClaw 的SKILL搞得头大啊！

## 核心主旨
推薦一個 GitHub 上的精選 OpenClaw Skills 列表 `Awesome-openclaw-skills`，旨在解決現有 Skills 混亂、品質不一的問題，幫助用戶打造更強大的 OpenClaw 助手。

## 問題點
*   OpenClaw Skills 混亂、垃圾內容多、品質參差不齊。
*   難以找到可靠的 Skills 來執行實際任務。

## 解決方案 (推薦 `Awesome-openclaw-skills`)
*   **寶藏列表**：在 GitHub 上擁有 1.8K ⭐️，是專為 OpenClaw 整理的 Skills 精選合集。
*   **嚴格篩選**：作者從 ClawHub 上總計 5705 個社區技能中，嚴格篩選出 3002 個高品質、無垃圾、無重複、無惡意的內容。
*   **清晰分類**：整理成 30 多個清晰類別。

## 覆蓋範圍廣泛 (幾乎涵蓋所有場景)
*   Coding Agents、多代理協同、IDE 集成。
*   GitHub PR 自動合併測試、DevOps 雲基礎設施管理。
*   WhatsApp / Gmail / 日曆完整集成。
*   瀏覽器無頭自動化、智能家居、Tesla 車輛控制。
*   語音合成轉錄、PDF OCR 處理、圖像視頻生成。
*   隱私搜索、持久記憶系統、安全防護。
*   Agent 間通信協議、生產力工具、個人發展助手等。

## 安裝方式極簡
*   一條命令：`npx clawhub@latest install <skill-slug>`。
*   或手動將 Skill 目錄放入 `~/.openclaw/skills/`。

## Skill 特點
*   每個 Skill 都附有詳細的 `SKILL.md` 使用說明。
*   全部本地優先，確保隱私安全。
*   擴展性極強。

## 強烈推薦
*   對於正在使用本地 AI Agent、想將助手打造成全能生產力工具的朋友來說，這份列表是「降維打擊級」的神器。
